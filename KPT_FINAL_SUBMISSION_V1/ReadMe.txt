JDK Version notes

-----------------


The JavaMail API supports JDK 1.5 or higher.
  
Note that we have
currently tested this implementation with JDK 1.7.





Installation

------------
1. Download & extract the Group8 zip file.

2. Compile SentimentAnalysisV1.java
  javac SentimentAnalysisV1.java

3. Now run the SentimentAnalysisV1 java program
   java SentimentAnalysisV1




Additional Information

------------------------
1. We have compiled and executed the SentimentAnalysisV1.java in JGrasp.
2. There are no dependencies for our project.

3. File Not Found Exception - If the file path for classBAYES,tweetsBAYES,stopwords and testData is not set properly.
